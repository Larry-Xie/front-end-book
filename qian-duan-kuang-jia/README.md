# 前端框架

## 一、引言

现在前后端分离已经逐渐成为趋势，并且前端开发也越来越复杂，应运而生的就是前端框架。而前端框架每年也层出不穷，日新月异，但是主流的框架也就大家所熟知的 Vue、React、Angular。

对于各个前端框架，虽然看上去不尽相同，但是它们的核心思想和一些原理却有着异曲同工之处。也是我们需要了解的，这样对于框架的对比以及选型才能更充分。

框架通常包含的方面有：

* MVVM 模型
* 组件化
* 渲染机制
* 状态管理
* 路由机制
* CSS 方案
* 构建工具
* SSR

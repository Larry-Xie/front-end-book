# 全栈进阶

## 一、引言

虽然现在是前后端分离的大趋势，但是不管是前端开发还是后端开发，都需要具备一定的全局观，尤其是对于前端开发而言，不能偏居一隅。虽不求对于后端语言的掌握，但是对于系统设计和架构，需要有所了解和认知。\

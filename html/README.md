# HTML

## 一、引言

HTML 的考点其实并不是很多，基本都是一些基础知识：

* HTML 语义化
* HTML 元数据
* HTML 5 新特性
* SVG 和 Canvas
* 预加载和异步加载

还有一些不常用但是是 W3C HTML 标准里的一些内容，需要知道：

* Web Worker
* Web Sockets
* Web Component

其中预加载和异步加载相对重要，需要熟记。
